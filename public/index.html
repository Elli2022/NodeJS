Node.js microservice built with Express, filesystem persistence, Mocha tests, and progressive refactoring branches. This site is the public documentation homepage for the repository.

Node.js Express TypeScript Mocha Filesystem
View on GitHub Related Next.js app

The REST API runs locally when you start the service (npm start). Netlify hosts this documentation page only — use the commands below to exercise /health and /api/v1/ on your machine.

API endpoints

  • GET /health — service status
  • GET /api/v1/ — list users
  • POST /api/v1/ — create user

Run locally

npm install
NODE_HOSTNAME=127.0.0.1 NODE_PORT=3000 NODE_NAME=nodejs-ms npm start

Example request

curl http://127.0.0.1:3000/health

curl -X POST http://127.0.0.1:3000/api/v1/ \
  -H "content-type: application/json" \
  -d '{"username":"demo123","password":"secret123"}'
